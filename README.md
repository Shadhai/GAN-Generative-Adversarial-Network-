# 🧠 Deep Learning & Generative AI Lab

A curated collection of **deep learning experiments, neural network architectures, and generative models** implemented from scratch and using modern frameworks.

This repository demonstrates hands-on implementation of **CNNs, DNNs, GANs, and self-attention-based models** across classification, regression, and image generation tasks.

---

## 📌 Repository Overview

This project serves as a **learning and experimentation hub** for core deep learning concepts, including:

* Supervised learning (classification & regression)
* Deep neural network optimization techniques
* Convolutional neural networks for attribute recognition
* Generative adversarial networks (GANs)
* Self-attention mechanisms for improved generation quality
* Image processing fundamentals

---

## 🧪 Projects Included

### 🖼️ Image Processing Fundamentals

Foundational techniques for preprocessing, transformations, and feature extraction in images.

---

### 🧠 CNN for Attribute Classification

A convolutional neural network designed to extract spatial features and classify image attributes effectively.

---

### 📊 Multi-Class Classification using ANN

A fully connected neural network built for solving multi-class classification problems using structured datasets.

---

### 📈 Regression using ANN (California Housing Dataset)

A deep learning regression model predicting housing prices using the California Housing dataset.

---

### ⚡ DNN for Multi-Label Classification

A deep neural network designed to handle multiple simultaneous output labels for complex classification tasks.

---

### 🛡️ DNN with Early Stopping & Regularization

Improved DNN training pipeline using:

* Early stopping
* Dropout
* L2 regularization
  to prevent overfitting and improve generalization.

---

### 🎨 GAN – Face Generation

A Generative Adversarial Network trained to generate realistic human face images from noise vectors.

---

### 🔥 Conditional GAN

A GAN variant that generates images conditioned on class labels for controlled generation.

---

### ⚙️ GAN Loss Functions

Experimental study and implementation of different GAN loss functions for stable training and improved convergence.

---

### 🧬 Self-Attention GAN

An advanced GAN architecture incorporating self-attention mechanisms to capture long-range dependencies in image generation.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/deep-learning-lab.git
cd deep-learning-lab
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run any project:

```bash
cd "GAN Face Generation"
python train.py
```
